# Basketball-training-app
 Đề tài tập trung xây dựng một ứng dụng di động hỗ trợ tập luyện bóng rổ và thể chất cá nhân Hệ thống sử dụng python ( FastApi) để quản lý cơ sở dữ liệu bài tập và lộ trình, kết hợp với ứng dụng Mobile giúp người dùng xem video hướng dẫn, theo dõi lịch tập hằng ngày và ghi lại tiến độ hoàn thành. Mục tiêu chính là số hóa quy trình tập luyện.
